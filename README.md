
# Carty

This is a grocery shopping list app. It's react native and uses expo.


## Authors

- [@Carlogy](https://www.github.com/carlogy)


## Run Locally

Clone the project

```bash
  git clone https://github.com/carlogy/Carty.git
```

Go to the project directory

```bash
  cd carty
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
